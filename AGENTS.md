# Project overview

This is a web app that sends Instapaper articles to Kindle and manages Instapaper articles

## Features

- List Instapaper articles
- Send Instapaper articles to Kindle

## Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- postmark for sending emails
- instapaper API for fetching and managing articles
